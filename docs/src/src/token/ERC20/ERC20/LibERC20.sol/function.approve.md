# approve
[Git Source](https://github.com/maxnorm/Compose/blob/f412c7c7d10aaefbcfad4abc92cd2d16bf0bacad/src/token/ERC20/ERC20/LibERC20.sol)

Approves a spender to transfer tokens on behalf of the caller.

Sets the allowance for the spender.


```solidity
function approve(address _spender, uint256 _value) ;
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`_spender`|`address`|The address to approve for spending.|
|`_value`|`uint256`|The amount of tokens to approve.|
