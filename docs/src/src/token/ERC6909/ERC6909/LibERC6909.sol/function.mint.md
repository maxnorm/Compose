Mints `_amount` of token id `_id` to `_to`.


```solidity
function mint(address _to, uint256 _id, uint256 _amount) ;
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|

|`_id`|`uint256`|The id of the token.|
|`_amount`|`uint256`|The amount of the token.|
