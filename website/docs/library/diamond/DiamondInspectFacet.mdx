<DocSubtitle>
Inspect diamond facets and their selectors
</DocSubtitle>

<Callout type="info" title="Key Features">
- Provides external functions for diamond introspection.
- Enables querying of facet addresses by function selector.
- Returns a list of all facets and their associated function selectors.
- `exportSelectors()` function aids in selector discovery for diamond upgrade mechanisms.
</Callout>

## Overview

This facet provides essential introspection capabilities for a diamond, allowing developers to query facet addresses and their associated function selectors. It exposes functions to retrieve the facet addresses, map function selectors to their respective facets, and list all facets along with their selectors. This enables dynamic interaction and understanding of the diamond's on-chain logic.

## Best Practices

<Callout type="tip" title="Best Practice">
- Call facet introspection functions through the diamond proxy interface.
- Use `facets()` to retrieve a comprehensive list of all facets and their selectors.
- Verify `facetAddress(_functionSelector)` returns a non-zero address before interacting with a specific function.
</Callout>

## Security Considerations

<Callout type="warning" title="Security">
This facet primarily provides read-only access to diamond configuration and does not handle state changes. Standard Solidity security practices should be followed when interacting with the diamond proxy.
</Callout>
