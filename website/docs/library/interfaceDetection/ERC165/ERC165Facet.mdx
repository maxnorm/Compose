Interface detection for Compose diamonds
</DocSubtitle>

<Callout type="info" title="Key Features">
- Implements ERC-165 interface detection standard.
- Provides `supportsInterface` for external queries.
- Exports facet selectors for diamond integration.
- Self-contained with no external dependencies.
</Callout>

## Overview

This facet implements ERC-165 interface detection for Compose diamonds. It exposes functions to query supported interfaces and exports its own selectors. Developers add this facet to enable standard interface identification within their diamond.

## Best Practices

<Callout type="tip" title="Best Practice">
- Ensure the ERC165Facet is added to the diamond during deployment.
- Call `supportsInterface` through the diamond proxy to check for interface support.
- Use `exportSelectors` during diamond deployment to discover facet selectors.
</Callout>

## Security Considerations

<Callout type="warning" title="Security">
Follow standard Solidity security practices. Input validation is handled by the `supportsInterface` function. Access to internal storage is managed via the diamond storage pattern.
</Callout>
