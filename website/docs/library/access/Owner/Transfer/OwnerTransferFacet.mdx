<DocSubtitle>
Manages ownership and provides selector discovery
</DocSubtitle>

<Callout type="info" title="Key Features">
- Manages diamond ownership via diamond storage.
- Provides an `exportSelectors` function for dynamic discovery.
- Follows EIP-2535 Diamond Standard for facet integration.
</Callout>

## Overview

This facet provides ownership management and selector export for Compose diamonds. It allows for transferring ownership and exposes a function to discover its selectors, facilitating diamond upgrades and integration. Ownership is managed via diamond storage.

## Best Practices

<Callout type="tip" title="Best Practice">
- Initialize ownership during diamond deployment.
- Use `transferOwnership` to safely change the contract owner.
- Call `exportSelectors` to discover the facet's selectors for diamond upgrades.
</Callout>

## Security Considerations

<Callout type="warning" title="Security">
The `transferOwnership` function is protected by access control, ensuring only the current owner can transfer ownership. Input validation is performed on `_newOwner`. Refer to Compose's access control facets for comprehensive role management.
</Callout>
