<DocSubtitle>
Manages role administration within a diamond
</DocSubtitle>

<Callout type="info" title="Key Features">
- Exposes `setRoleAdmin` for programmatic role hierarchy management.
- Provides `exportSelectors` for facet discovery.
- Functions are `external` for direct diamond proxy interaction.
- Utilizes diamond storage for role configuration.
</Callout>

## Overview

This facet provides administrative functions for managing roles within a Compose diamond. It allows setting the admin role for specific roles and exporting the facet's selectors. Developers integrate this facet to control role hierarchies and manage permissions programmatically.

## Best Practices

<Callout type="tip" title="Best Practice">
- Initialize role administration during diamond setup.
- Enforce that only designated admin roles can call `setRoleAdmin`.
- Use `exportSelectors` to dynamically discover and interact with facet functions.
</Callout>

## Security Considerations

<Callout type="warning" title="Security">
The `setRoleAdmin` function reverts with `AccessControlUnauthorizedAccount` if the caller is not the current admin of the role being modified. Input roles should be validated to ensure expected behavior.
</Callout>
