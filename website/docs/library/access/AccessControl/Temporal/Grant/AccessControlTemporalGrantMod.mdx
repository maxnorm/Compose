<DocSubtitle>
Grants roles with expiry using diamond storage
</DocSubtitle>

<Callout type="info" title="Key Features">
- Grants roles with a specified expiry timestamp.
- Functions are `internal`, designed for use within custom facets.
- Leverages the diamond storage pattern for shared state management.
- Emits a `RoleGrantedWithExpiry` event upon successful role granting.
</Callout>

<Callout type="info" title="Module Usage">
This module provides internal functions for use in your custom facets. Import it to access shared logic and storage.
</Callout>

## Overview

This module provides functions for granting roles with a specific expiry timestamp, utilizing the diamond storage pattern for shared state. Facets can integrate this module to manage time-bound permissions within a Compose diamond. Changes to role expiry are immediately reflected across all facets accessing the same diamond storage.

### grantRoleWithExpiry

Grants a role to an account with an expiry timestamp. Only the admin of the role can grant it with expiry. Emits a &#123;RoleGrantedWithExpiry&#125; event. Reverts with AccessControlUnauthorizedAccount If the caller is not the admin of the role.

## Best Practices

<Callout type="tip" title="Best Practice">
- Call `grantRoleWithExpiry` only when the caller has the necessary administrative privileges for the role.
- Ensure the `expiresAt` timestamp is set correctly to avoid premature role expiration or overly long durations.
- Handle the `AccessControlRoleExpired` and `AccessControlUnauthorizedAccount` errors returned by module functions.
</Callout>

## Integration Notes

<Callout type="success" title="Shared Storage">
This module interacts with diamond storage via the `ACCESS_CONTROL_STORAGE_POSITION`. It utilizes the `AccessControlStorage` and `AccessControlTemporalStorage` structs to manage role assignments and their expiry times. Functions are internal and designed to be called by facets that have access to the shared diamond storage, ensuring immediate visibility of changes across all composing facets.
</Callout>
