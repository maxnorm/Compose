<DocSubtitle>
Event emitted when a role is granted with an expiry timestamp.
</DocSubtitle>

<Callout type="info" title="Key Features">
- Grants roles with specific expiry timestamps, enabling time-bound permissions.
- Allows checking if a role assignment has expired using &#x60;isRoleExpired&#x60;.
- Provides a &#x60;requireValidRole&#x60; function to enforce non-expired role checks within facets.
- Supports revoking temporal roles explicitly via &#x60;revokeTemporalRole&#x60;.
</Callout>

<Callout type="info" title="Module Usage">
This module provides internal functions for use in your custom facets. Import it to access shared logic and storage.
</Callout>

## Overview

The AccessControlTemporalMod provides time-bound role management for your diamond. It allows granting roles that automatically expire, enhancing security and operational flexibility by ensuring permissions are temporary. This module integrates seamlessly with the diamond&#x27;s storage pattern, allowing facets to manage and query temporal role assignments.

## Usage Example

<ExpandableCode language="solidity" maxLines={20} title="Example Implementation">
{`pragma solidity ^0.8.30;

import {IAccessControlTemporalMod} from "./interfaces/IAccessControlTemporalMod.sol";

contract MyFacet {
    // Assume diamond storage is accessible and initialized
    // For example, using DiamondLoupeFacet or a similar mechanism
    IAccessControlTemporalMod internal accessControlTemporalMod;

    // Initialization function (e.g., called during diamond upgrade)
    function initialize(address _accessControlTemporalModAddress) external {
        accessControlTemporalMod = IAccessControlTemporalMod(_accessControlTemporalModAddress);
    }

    /**
     * @notice Grants a role to an account with a specific expiry timestamp.
     */
    function grantMyRole(address _account, bytes32 _role, uint64 _expiry) external {
        // Assuming you have access to the diamond's storage or a reference
        // to the AccessControlTemporalMod contract instance.
        accessControlTemporalMod.grantRoleWithExpiry(_role, _account, _expiry);
    }

    /**
     * @notice Revokes a temporal role from an account.
     */
    function revokeMyRole(address _account, bytes32 _role) external {
        accessControlTemporalMod.revokeTemporalRole(_role, _account);
    }

    /**
     * @notice Checks if a role assignment for an account has expired.
     */
    function checkRoleExpiry(address _account, bytes32 _role) external view returns (bool) {
        return accessControlTemporalMod.isRoleExpired(_role, _account);
    }

    /**
     * @notice Requires that an account has a valid, non-expired role.
     */
    function performActionRequiringRole(address _account, bytes32 _role) external {
        accessControlTemporalMod.requireValidRole(_role, _account);
        // Proceed with action...
    }
}`}
</ExpandableCode>

## Best Practices

<Callout type="tip" title="Best Practice">
- Ensure &#x60;grantRoleWithExpiry&#x60; is called with a sensible future timestamp to avoid immediate expiration.
- Utilize &#x60;requireValidRole&#x60; before executing sensitive operations to enforce temporal access controls.
- Implement robust error handling for &#x60;AccessControlUnauthorizedAccount&#x60; and &#x60;AccessControlRoleExpired&#x60; to provide informative feedback to users.
</Callout>

## Integration Notes

<Callout type="success" title="Shared Storage">
This module interacts with the diamond&#x27;s storage pattern. Facets can access its functionality through an instance of &#x60;IAccessControlTemporalMod&#x60;. The &#x60;grantRoleWithExpiry&#x60; and &#x60;revokeTemporalRole&#x60; functions directly modify the temporal role assignments within the diamond&#x27;s state. Access to the underlying storage for AccessControl and AccessControlTemporal can be queried via &#x60;getAccessControlStorage&#x60; and &#x60;getStorage&#x60; respectively, though direct manipulation of these is not recommended for facets. Invariants related to role validity and expiry are managed internally by this module.
</Callout>
