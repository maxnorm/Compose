## Overview

LibOwnerTwoSteps is a Solidity library designed to manage a two-step ownership transfer process in diamond contracts, adhering to the ERC-173 standard for ownership. It simplifies ownership management while ensuring security and preventing unauthorized access to critical operations.


[View Source](https://github.com/maxnorm/Compose/blob/7206f045d711880c211b967e80d72745b52f5cfc/src/access/OwnerTwoSteps/LibOwnerTwoSteps.sol)


<Callout type="info" title="Library Usage">
This library provides internal functions for use in your custom facets. Import it to access shared storage.
</Callout>

## Usage Example

```solidity
pragma solidity ^0.8.0;

import {LibOwnerTwoSteps} from &quot;path_to_your_lib/LibOwnerTwoSteps.sol&quot;;

contract ExampleDiamond {
    function transferDiamondOwnership(address _newOwner) external {
        LibOwnerTwoSteps.requireOwner(); // Ensures only the current owner can call
        LibOwnerTwoSteps.transferOwnership(_newOwner); // Initiates transfer
    }

    function acceptDiamondOwnership() external {
        LibOwnerTwoSteps.acceptOwnership(); // Allows the pending owner to finalize
    }

    function checkCurrentOwner() external view returns (address) {
        return LibOwnerTwoSteps.owner(); // Returns the current owner address
    }
}
```



## Best Practices

<Callout type="tip" title="Best Practice">
- Call LibOwnerTwoSteps.requireOwner() before invoking functions that require owner privileges.
- Use LibOwnerTwoSteps.transferOwnership() to initiate any ownership transfer securely and LibOwnerTwoSteps.acceptOwnership() to confirm it.
- Avoid skipping the two-step process as it provides a critical safeguard against accidental or unauthorized ownership changes.
</Callout>



## Integration Notes

<Callout type="success" title="Shared Storage">
LibOwnerTwoSteps operates using the diamond storage pattern, accessing the struct at a defined storage slot via inline assembly. Any state changes made by this library are immediately visible to facets that reference the same storage.
</Callout>
