## Overview

LibOwnerTwoSteps provides functions to manage ownership through a secure, two-step process in accordance with the ERC-173 standard. It ensures controlled and explicit ownership transfers in diamond architecture, making it valuable for permission management in decentralized applications.


[View Source](https://github.com/maxnorm/Compose/blob/38545161423dd99e31b48b0eff41d260712fe731/src/access/OwnerTwoSteps/LibOwnerTwoSteps.sol)


<Callout type="info" title="Library Usage">
This library provides internal functions for use in your custom facets. Import it to access shared storage.
</Callout>

## Usage Example

```solidity
pragma solidity ^0.8.0;

import &quot;path/to/LibOwnerTwoSteps.sol&quot;;

contract ExampleDiamond {

    function initiateOwnershipTransfer(address _newOwner) external {
        LibOwnerTwoSteps.transferOwnership(_newOwner);
    }

    function finalizeOwnershipTransfer() external {
        LibOwnerTwoSteps.acceptOwnership();
    }

    function getCurrentOwner() external view returns (address) {
        return LibOwnerTwoSteps.owner();
    }

}
```



## Best Practices

<Callout type="tip" title="Best Practice">
- Ensure secure implementation of authorization checks to limit illegal writes or transfers.
- Always finalize the transferOwnership process using the acceptOwnership function.
- Use getStorage or direct diamond storage patterns cautiously for on-chain operations to maintain integrity.
</Callout>
