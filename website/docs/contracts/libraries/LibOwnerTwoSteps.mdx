## Overview

ERC-173 Two-Step Contract Ownership Library - Provides two-step ownership transfer logic for facets or modular contracts.


[View Source](https://github.com/maxnorm/Compose/blob/057043a659d1a2c7cf32c80148e04c3e2c6d85f2/src/access/OwnerTwoSteps/LibOwnerTwoSteps.sol)


<Callout type="info" title="Library Usage">
This library provides internal functions for use in your custom facets. Import it to access shared storage.
</Callout>


## Storage


Storage position: `OWNER_STORAGE_POSITION` - Used for diamond storage pattern.
