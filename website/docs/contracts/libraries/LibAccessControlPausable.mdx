## Overview

LibAccessControlPausable is a Solidity library designed to integrate access control with pausability features in diamond contracts. It enables roles to be paused or unpaused, providing fine-grained control over functionality while maintaining seamless compatibility with diamond storage.


[View Source](https://github.com/maxnorm/Compose/blob/7206f045d711880c211b967e80d72745b52f5cfc/src/access/AccessControlPausable/LibAccessControlPausable.sol)


<Callout type="info" title="Library Usage">
This library provides internal functions for use in your custom facets. Import it to access shared storage.
</Callout>

## Usage Example

```solidity
pragma solidity ^0.8.0;

import &quot;./LibAccessControlPausable.sol&quot;;

contract ExampleFacet {
    function pauseAdminRole() external {
        bytes32 role = keccak256(&quot;ADMIN_ROLE&quot;);
        LibAccessControlPausable.pauseRole(role);
    }

    function executeAdminAction() external {
        bytes32 role = keccak256(&quot;ADMIN_ROLE&quot;);
        LibAccessControlPausable.requireRoleNotPaused(role, msg.sender);
        // Perform role-specific action here
    }

    function isPaused(bytes32 role) external view returns (bool) {
        return LibAccessControlPausable.isRolePaused(role);
    }
}
```



## Best Practices

<Callout type="tip" title="Best Practice">
- Always verify role pausability using `isRolePaused` before executing critical operations.
- Ensure `requireRoleNotPaused` is used in sensitive areas to enforce role and pause checks.
- Align external storage structs for reuse and maintain structural integrity per diamond storage patterns.
</Callout>

LibAccessControlPausable adheres to the diamond storage pattern by using explicit, reusable storage slots for pausable access control functionalities. Any state changes made through this library are immediately accessible to all facets, ensuring consistent behavior across the diamond.
</Callout>
