## Overview

LibAccessControl provides robust role-based access management for diamond proxy contracts, ensuring granular permissions for accounts within the Compose ecosystem. It helps maintain secure and composable control over contract functionality by managing roles and administrators efficiently.


[View Source](https://github.com/maxnorm/Compose/blob/e3e377cf3d77e94dc30e10812da2bdd1907ca159/src/access/AccessControl/LibAccessControl.sol)


<Callout type="info" title="Library Usage">
This library provides internal functions for use in your custom facets. Import it to access shared storage.
</Callout>

## Usage Example

```solidity
pragma solidity ^0.8.0;

import &quot;LibAccessControl.sol&quot;;

contract ExampleDiamondFacet {

  function exampleFunction(address _account) external {
    // Check if the caller has the required role
    LibAccessControl.requireRole(keccak256(&quot;EXAMPLE_ROLE&quot;), _account);
    
    // Do something if the account has the required role
  }

  function grantRole(address _account) external {
    // Grant a role to an account
    LibAccessControl.grantRole(keccak256(&quot;EXAMPLE_ROLE&quot;), _account);
  }

  function revokeRole(address _account) external {
    // Revoke a role from an account
    LibAccessControl.revokeRole(keccak256(&quot;EXAMPLE_ROLE&quot;), _account);
  }
}
```



## Best Practices

<Callout type="tip" title="Best Practice">
- Always opt for custom errors (e.g., AccessControlUnauthorizedAccount) for clarity and gas efficiency.
- Follow proper role administration by securely managing who holds role privileges.
- Regularly audit role assignments to ensure only necessary permissions are granted.
</Callout>



## Integration Notes

<Callout type="success" title="Shared Storage">
LibAccessControl integrates seamlessly with the diamond storage pattern to ensure consistent visibility of role and admin state changes across all facets. Any modifications made using this library are immediately shared and accessible by other facets within the diamond contract.
</Callout>
