## Overview

LibERC721Enumerable is a library designed to facilitate the storage and management of ERC-721 token enumeration data within a diamond proxy architecture. It provides robust functions for token transfers, minting, and burning while maintaining enumeration integrity.


[View Source](https://github.com/maxnorm/Compose/blob/7206f045d711880c211b967e80d72745b52f5cfc/src/token/ERC721/ERC721Enumerable/LibERC721Enumerable.sol)


<Callout type="info" title="Library Usage">
This library provides internal functions for use in your custom facets. Import it to access shared storage.
</Callout>

### mint


Mints a new ERC-721 token to the specified address, adding it to enumeration lists. Reverts if the receiver address is zero or if the token already exists.




## Errors


### ERC721NonexistentToken


Thrown when attempting to interact with a non-existent token.



```solidity
error ERC721NonexistentToken(uint256 _tokenId);
```



### ERC721IncorrectOwner


Thrown when the sender is not the owner of the token.



```solidity
error ERC721IncorrectOwner(address _sender, uint256 _tokenId, address _owner);
```



### ERC721InvalidSender


Thrown when the sender address is invalid.



```solidity
error ERC721InvalidSender(address _sender);
```



### ERC721InvalidReceiver


Thrown when the receiver address is invalid.



```solidity
error ERC721InvalidReceiver(address _receiver);
```



### ERC721InsufficientApproval


Thrown when an operator lacks approval to manage a token.



```solidity

```

## Best Practices

<Callout type="tip" title="Best Practice">
- Always validate ownership and approvals when using transferFrom.
- Ensure receiver addresses are non-zero during minting to prevent accidental token loss.
- Use the provided getStorage function to access the correct storage slot for enumeration data.
</Callout>



## Integration Notes

<Callout type="success" title="Shared Storage">
LibERC721Enumerable operates with diamond storage, ensuring all updates to the storage slot are immediately visible across interacting facets. Accessing enumeration details or modifying token data through this library adheres to the Compose library&#39;s storage design principles.
</Callout>
