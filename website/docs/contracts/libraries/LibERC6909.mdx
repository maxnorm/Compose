## Functions


### getStorage


Returns a pointer to the ERC-6909 storage struct. Uses inline assembly to access the storage slot defined by STORAGE_POSITION.






### mint


Mints `_amount` of token id `_id` to `_to`.






### burn


Burns `_amount` of token id `_id` from `_from`.






### transfer


Transfers `_amount` of token id `_id` from `_from` to `_to`. Allowance is not deducted if it is `type(uint256).max` Allowance is not deducted if `_by` is an operator for `_from`.






### approve


Approves an amount of an id to a spender.






### setOperator


Sets or removes a spender as an operator for the caller.
