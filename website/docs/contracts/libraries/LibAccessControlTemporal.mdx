## Overview

LibAccessControlTemporal provides functions for managing time-bound role-based access control in diamond contract systems. It enables role assignments with expiry timestamps, ensuring flexible and secure access control mechanisms within the diamond storage pattern.


[View Source](https://github.com/maxnorm/Compose/blob/38545161423dd99e31b48b0eff41d260712fe731/src/access/AccessControlTemporal/LibAccessControlTemporal.sol)


<Callout type="info" title="Library Usage">
This library provides internal functions for use in your custom facets. Import it to access shared storage.
</Callout>

## Usage Example

```solidity
pragma solidity ^0.8.0;

import {LibAccessControlTemporal} from &quot;./LibAccessControlTemporal.sol&quot;;

contract ExampleDiamondFacet {
    function grantRoleWithExpiration(bytes32 _role, address _account, uint256 _expiry) external {
        LibAccessControlTemporal.grantRoleWithExpiry(_role, _account, _expiry);
    }

    function checkRoleValidity(bytes32 _role, address _account) external view {
        LibAccessControlTemporal.requireValidRole(_role, _account);
    }

    function revokeExpiredRole(bytes32 _role, address _account) external {
        if (LibAccessControlTemporal.isRoleExpired(_role, _account)) {
            LibAccessControlTemporal.revokeTemporalRole(_role, _account);
        }
    }
}
```



## Best Practices

<Callout type="tip" title="Best Practice">
- Use `requireValidRole` to enforce that accounts interacting with sensitive functionality possess a valid, non-expired role.
- Regularly call `isRoleExpired` before allowing critical operations to ensure expired roles are not unintentionally used.
- Plan the diamond storage layout carefully to maintain compatibility across facets.
</Callout>



## Integration Notes

<Callout type="success" title="Shared Storage">
LibAccessControlTemporal leverages the diamond storage pattern, ensuring state is consistently shared and updated across facets. Changes made to roles and expirations through this library are immediately visible to all diamond facets leveraging this storage.
</Callout>
