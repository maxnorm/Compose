## Overview

LibERC165 — ERC-165 Standard Interface Detection Library - Provides internal functions and storage layout for ERC-165 interface detection.


[View Source](https://github.com/maxnorm/Compose/blob/f648de11ffa3a1ed910367f4bea22f30eb1dac06/src/interfaceDetection/ERC165/LibERC165.sol)


<Callout type="info" title="Library Usage">
This library provides internal functions for use in your custom facets. Import it to access shared storage.
</Callout>
