## Overview

This section contains auto-generated API documentation for all Compose facets. Facets are self-contained smart contract implementations that can be added to your diamond.

The documentation is generated from NatSpec comments in the Solidity source code using the `forge doc` command, then converted to Docusaurus format and enhanced with AI-generated content.

## How Documentation is Generated

1. **NatSpec Extraction**: When Solidity files in `src/` are modified, `forge doc` extracts NatSpec comments
2. **Conversion**: The markdown is converted to Docusaurus MDX with custom components
3. **Enhancement**: GitHub Copilot adds usage examples and best practices
4. **Review**: A PR is created for human review before publishing
