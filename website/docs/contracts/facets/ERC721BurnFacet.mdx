## Overview

A complete, dependency-free ERC-721 implementation using the diamond storage pattern.

This facet provides metadata, ownership, approvals, safe transfers, minting, burning, and helpers.


[View Source](https://github.com/maxnorm/Compose/blob/e3e377cf3d77e94dc30e10812da2bdd1907ca159/src/token/ERC721/ERC721/ERC721BurnFacet.sol)



<Callout type="info" title="Key Features">
- Self-contained facet with no imports or inheritance
- Only `external` and `internal` function visibility
- Follows Compose readability-first conventions
- Ready for diamond integration
</Callout>
