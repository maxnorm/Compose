## Overview

**Title:**

ERC-2981 NFT Royalty Standard Interface

## Functions

### royaltyInfo

```solidity
function royaltyInfo(uint256 _tokenId, uint256 _salePrice)
    external
    view
    returns (address receiver, uint256 royaltyAmount);
```

Returns royalty information for a given token and sale price. Called with the sale price to determine how much royalty is owed and to whom. Implementations MUST calculate royalty as a percentage of the sale price.
